# Combined Algorithm Dataset

- Problems: 339
- Languages: C, Java, Rust
- Layout: one Markdown file per problem per language in `c/`, `java/`, and `rust/`.
- Tests: shared JSON files in `tests/`.
- Source attribution: problems with a clear external origin include a `source` field in `tests/*.json` and `manifest.json`.
- Generation: `python3 scripts/combine_datasets.py`
