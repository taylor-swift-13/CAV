# Array Equal

## 问题描述

实现一个函数，输入两个长度都为 `n` 的整数数组 `a` 和 `b`，判断它们是否逐项相等。

约定：

- `n >= 0`
- 两个数组长度都恰好是 `n`
- 函数不修改数组
- 如果对所有下标 `i` 都有 `a[i] == b[i]`，返回 `1`
- 否则返回 `0`

## 参考实现

```rust
pub fn array_equal(n: i32, a: &[i32], b: &[i32]) -> i32 {
    let mut i: i32;

    i = 0;
    while i < n {
        if a[(i) as usize] != b[(i) as usize] {
            return 0;
        }
        i += 1;
    }

    return 1;
}
```

## 说明

这道题适合验证“双数组同步扫描”模式：

- 两个输入数组
- 只读比较
- 发现不等后立即返回
