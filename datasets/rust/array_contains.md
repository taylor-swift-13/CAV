# Array Contains

## 问题描述

实现一个函数，输入长度为 `n` 的整数数组 `a` 和整数 `k`，判断数组中是否存在等于 `k` 的元素。

约定：

- `n >= 0`
- 数组长度恰好是 `n`
- 函数不修改数组
- 若存在某个下标 `i` 满足 `a[i] == k`，返回 `1`
- 否则返回 `0`

## 参考实现

```rust
pub fn array_contains(n: i32, a: &[i32], k: i32) -> i32 {
    let mut i: i32;

    i = 0;
    while i < n {
        if a[(i) as usize] == k {
            return 1;
        }
        i += 1;
    }

    return 0;
}
```

## 说明

这道题适合验证最基础的“扫描并返回布尔结果”模式：

- 单层循环
- 只读数组
- 命中后立即返回
