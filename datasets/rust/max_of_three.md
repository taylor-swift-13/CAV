# Max Of Three

## 问题描述

实现一个函数，输入三个整数 `a`、`b`、`c`，返回其中的最大值。

约定：

- 输入都是普通整数
- 返回值必须等于三个输入中的一个
- 返回值不小于任意输入

## 参考实现

```rust
pub fn max_of_three(a: i32, b: i32, c: i32) -> i32 {
    let mut m: i32 = a;

    if b > m {
        m = b;
    }
    if c > m {
        m = c;
    }

    return m;
}
```

## 说明

这是最简单的三数最大值题，适合验证“分支更新 + 最值语义”。
